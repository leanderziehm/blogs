| new window               | Ctrl + b + c         |
| view sessions            | Ctrl + b + s         |
| select pane              | Ctrl + b + 1,2,3 ... |
| copy mode                | Ctrl + b + \[        |
| keys                     | Ctrl + b + ?         |
| close windows and panes  | exit                 |
