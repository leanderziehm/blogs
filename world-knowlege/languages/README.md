# Languages

```
Human Languages
│
├── Indo-European (Eurasia)
│   ├── Germanic → English, German, Dutch, Swedish
│   ├── Romance → Spanish, French, Italian, Portuguese, Romanian
│   ├── Indo-Iranian → Hindi, Urdu, Persian, Bengali
│   ├── Slavic → Russian, Polish, Czech, Serbian
│   └── Celtic → Irish, Welsh, Scottish Gaelic
│
├── Sino-Tibetan (East Asia)
│   ├── Sinitic → Mandarin, Cantonese, Wu
│   └── Tibeto-Burman → Burmese, Tibetan
│
├── Afroasiatic (North Africa / Middle East)
│   ├── Semitic → Arabic, Hebrew, Amharic
│   ├── Berber → Tamazight languages
│   └── Egyptian (ancient → Coptic)
│
├── Niger-Congo (Sub-Saharan Africa)
│   ├── Bantu → Swahili, Zulu, Shona
│   └── Other Niger-Congo → Yoruba, Igbo
│
├── Austronesian (Pacific / Southeast Asia)
│   ├── Malayo-Polynesian → Malay, Indonesian, Tagalog
│   └── Oceanic → Hawaiian, Māori, Samoan
│
├── Dravidian (South Asia)
│   ├── South Dravidian → Tamil, Malayalam
│   ├── Central Dravidian → Telugu
│   └── North Dravidian → Brahui
│
├── Uralic (Northern Eurasia)
│   ├── Finno-Ugric → Finnish, Hungarian, Estonian
│   └── Samoyedic → Nenets
│
└── Turkic (Central Eurasia)
    ├── Oghuz → Turkish, Azerbaijani
    ├── Kipchak → Kazakh, Tatar
    └── Karluk → Uzbek, Uyghur
```
