# Networking

| Layer                 | Purpose                                                    | Protocols / Technologies                                                                                                                                                           |
| --------------------- | ---------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Application Layer** | User-facing network services and application communication | BGP, DHCP (v6), DNS, FTP, HTTP (HTTP/3), HTTPS, IMAP, IPP, IRC, LDAP, MGCP, MQTT, NNTP, NTP, OSPF, POP3, PTP, ONC/RPC, RTP, RTSP, RIP, SIP, SMTP, SNMP, SSH, Telnet, TLS/SSL, XMPP |
| **Transport Layer**   | End-to-end communication between hosts                     | TCP, UDP, DCCP, SCTP, RSVP, QUIC                                                                                                                                                   |
| **Internet Layer**    | Logical addressing and routing across networks             | IPv4, IPv6, ICMP (v6), NDP, ECN, IGMP, IPsec                                                                                                                                       |
| **Link Layer**        | Physical network access and local delivery                 | ARP, Tunnels (e.g., GRE/IP-in-IP), PPP, MAC                                                                                                                                        |
